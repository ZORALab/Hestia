Objective

HestiaHUGO's robots HTML meta template rendering partial function aims to render all page-specific .Robots data structure as HTML meta output.

This is primarily for HTML codes in the <head> section.

Algorithm

Here are a summary of how this template function works.

Input

This template partial function takes .Robots and renders them into HTML Meta tag like <meta name="..." content="..."> for each robot data.

Processing

This partial function operates in the following steps:

  1. Render each .Robots data into <meta name="{{- safeHTML .Name -}}" content="{{- safeHTML .Content -}}"> for each robot data. Both .Name and .Content are secured with Hugo's safeHTML function.

Output

Robots HTML meta codes are rendered out.
