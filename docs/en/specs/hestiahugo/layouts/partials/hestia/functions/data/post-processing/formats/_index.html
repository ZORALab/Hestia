Objective and Purposes

The primary objective is to process all Hestia's Page Formats data into its final form, ready for use.

How It Works

By default, the algorithm runs through each .Format data in the .Page.Formats data list. It sends all its elements to }}">Hestia Trim Data Whitespace partial function for clean up. It also process .Format.Protocol default value against .URL.Base and if all else fails, as https.

Input

This partial function accepts the following inputs:

  • . - required Hestia Page data structure.

Working Algorithm

This partial function works in the following manner:

  1. Process .Formats.List data list.

  2. Process .Formats.Alternatives data list.

  3. Process .Formats.Current data.

  4. For each of the .Format data above:

    • Process whitespace clean up with: }}">Hestia Trim Data Whitespace partial function.

    • Process .Format.Protocol by matching its absense to .URL.Base and if all else fails, to https:// as last resort.

Output

This function returns all the following possible outputs:

  • OK - The data has been successfully processed.
