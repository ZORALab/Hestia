Objective and Purposes

The primary objective is to systematically and uniformly renders the page keywords according to a given pattern.

This is primarily used for HTML codes in <head> section.

How It Works

By default, this function renders the .Keywords into <meta name="keywords" content="..."> HTML Code.

Input

This partial function accepts the following inputs:

  • .Page - required Hestia Page data structure.

Working Algorithm

This partial function works in the following manner:

  1. Render the .Keywords into HTML format when the list is not empty.

Output

This function returns all the following possible outputs:

  • OK - HTML title code rendered with values.
