Objective and Purposes

The primary objective is to present a readable and practical warning messages into the console log.

How It Works

By default, this function obtains and pinpoint the source of the problem info and format it alongside the actual message (.Console.Message). Then, it renders its formatted message to the console.

Input

This partial function accepts the following inputs:

  • .Page - required Hestia Page data structure.

  • .Console.Message - required warning message.

Working Algorithm

This partial function works in the following manner:

  1. Obtain the necessary page info for rectifications like page location, shortcode causes, and file location whenever available alongside error message.

  2. Format the output string and render to console with Hugo's warnf.

Output

This function returns all the following possible outputs:

  • OK - The message is rendered to console. It should be something as such: {{% highlight bash "linenos=table,hl_lines=,linenostart=1" %}} WARN 2022/07/19 09:23:54 On Page : en/specs/hestiahugo/layouts/partials/hestia/functions/console/warningf/ File : specs/hestiahugo/layouts/partials/hestia/functions/console/Warningf/_index.html Shortcode : Hestia/url ═══════════ Some kind of messages ═══════════ {{% /highlight %}}

  • ERROR - non-hestia structure - given input data structure is not Hestia data structure. The message is still rendered to console but the root cause identifications shall ceased to work. It should be something as such: {{% highlight bash "linenos=table,hl_lines=,linenostart=1" %}} WARN 2022/07/19 09:39:34 [ ERROR ] GIVEN DATA TO WarningF IS NOT HESTIA STRUCTURED!! ══════════════════════ Some kind of messages ══════════════════════ {{% /highlight %}}

Usage

To use such console output, assuming the Page context is a processed Hestia data structure, here is an example:

{{% highlight html "linenos=table,hl_lines=,linenostart=1" %}} {{- $Page := . -}} {{- $ret := printf "Some kind of messages\n" -}} {{- $ret = dict "Console" (dict "Message" $ret) -}} {{- partial "Hestia/functions/console/Warningf" (merge $Page $ret) -}} {{% /highlight %}}
