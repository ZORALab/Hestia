Design

{{- .Data.Page.Component.ID -}} UI component was primarily designed for citing a statement from an author. The idea is to make sure the W3C compatible <blockquote> syntax is styled and working natively as expected without having additional CSS classing, Javascript, or WASM to intervene.

Currently, the rendered design is as follows:

The scientists of today think deeply instead of clearly. One must be sane to think clearly, but one can think deeply and be quite insane. Nikolai Tesla

HTML

ZORALab's Hestia employs the W3C native syntax for simplicity and maximum compatibility sakes.

For reference, ZORALab's Hestia recommends the following minimal HTML codes for using this UI component: 

<blockquote>
	The scientists of today think deeply instead of clearly.
	One must be sane to think clearly, but one can think
	deeply and be quite insane.

	<cite>Nikolai Tesla BLA</cite>
</blockquote>

This will produce the result shown above. Please take note that the <cite> field is entirely optional but available.
