Design

{{- .Data.Page.Component.ID -}} UI component was primarily designed for tabulating a list of navigational materials like a list of anchored link to ease reader's ability to jump towards a section of interest. The idea is to keep W3C compatible <nav> syntax styled and working natively as expected without having complicated CSS classing, Javascript, or WASM to intervene.

Moreover, to ensure this component does not limits the creativity of the developer, the <nav> container is merely a layout styling while letting one to fills in any sensible contents with freedom.

HTML

ZORALab's Hestia employs the W3C native syntax for simplicity and maximum compatibility sakes.

For reference, ZORALab's Hestia recommends the following minimal HTML codes for using this UI component: 

<nav class='toc'>
	...
</nav>

This will produce the result shown above.

Please take note that the .toc CSS class is required to denotes the navigation container as a table of content.
