CSS

ZORALab's Hestia heavily rely on CSS to style this component and offering its css variables for customizations. Below are the list of available CSS variables at your disposal.

Display

Affects the display mode of the rendered component.

VARIABLE     : --divider-display
CSS PROPERTY : display
DEFAULT      : block (>= v1.2.0)

Margin

Affects the margin of the rendered component.

VARIABLE     : --divider-margin
CSS PROPERTY : margin
DEFAULT      : 0 auto (>= v1.2.0)

Padding

Affects the padding of the rendered component.

VARIABLE     : --divide-padding
CSS PROPERTY : padding
DEFAULT      : 0 (>= v1.2.0)

Border

Affects the border of the rendered component.

VARIABLE     : --divider-border
CSS PROPERTY : border
DEFAULT      : initial (>= v1.2.0)

Border Top

Affects the top border of the rendered component.

VARIABLE     : --divider-border-top
CSS PROPERTY : border-top
DEFAULT      : initial (>= v1.2.0)

Border Bottom

Affects the bottom border of the rendered component.

VARIABLE     : --divider-border-bottom
CSS PROPERTY : border-bottom
DEFAULT      : 1px solid var(--divider-color) (>= v1.2.0)

Min-Width

Affects the minimum width of the rendered component.

VARIABLE     : --divider-min-width
CSS PROPERTY : min-width
DEFAULT      : initial (>= v1.2.0)

Width

Affects the width of the rendered component.

VARIABLE     : --divider-width
CSS PROPERTY : width
DEFAULT      : 100% (>= v1.2.0)

Max-Width

Affects the maximum width of the rendered component.

VARIABLE     : --divider-max-width
CSS PROPERTY : max-width
DEFAULT      : 100% (>= v1.2.0)

Min-Height

Affects the minimum height of the rendered component.

VARIABLE     : --divider-min-height
CSS PROPERTY : min-height
DEFAULT      : initial (>= v1.2.0)

Height

Affects the height of the rendered component.

VARIABLE     : --divider-height

DEFAULT      : 1px (>= v1.2.0)

Max-Height

Affects the maximum height of the rendered component.

VARIABLE     : --divider-max-height

DEFAULT      : initial (>= v1.2.0)

Color

VARIABLE     : --divider-color

DEFAULT      : var(--color-primary-300) (>= v1.2.0)

Color (Inverted Mode)
